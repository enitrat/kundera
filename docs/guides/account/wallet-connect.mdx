Connect to browser wallets (ArgentX, Braavos) for user-facing applications. Kundera provides utilities to detect, connect, and manage wallet connections.

## Connect Wallet

```typescript
import { connectWallet } from 'kundera/wallet';

const { result, error } = await connectWallet();

if (error) {
  console.error('Connection failed:', error.message);
} else {
  console.log('Connected:', result.address);
  console.log('Chain:', result.chainId);
}
```

## Specify Wallet Type

Request a specific wallet:

```typescript
const { result } = await connectWallet({
  wallet: 'argentX', // or 'braavos'
});
```

| Wallet | Value |
|--------|-------|
| ArgentX | `'argentX'` |
| Braavos | `'braavos'` |
| Any | `undefined` (auto-detect) |

## Create WalletAccount

Convert a wallet connection to a usable account:

```typescript
import { connectWallet } from 'kundera/wallet';
import { connectAccountFromWallet } from 'kundera/account';
import { createHttpProvider } from 'kundera/provider';

// 1. Connect wallet
const { result: connection } = await connectWallet();

// 2. Create provider
const provider = createHttpProvider({
  url: 'https://starknet-mainnet.public.blastapi.io',
});

// 3. Create account from wallet
const account = await connectAccountFromWallet(connection.provider, provider);

// 4. Use account
const tx = await account.execute([...]);
```

## Listen for Account Changes

React to user switching accounts:

```typescript
import { onAccountChange } from 'kundera/wallet';

const unsubscribe = onAccountChange((event) => {
  console.log('Account changed to:', event.address);
  // Update your app state
});

// Later: stop listening
unsubscribe();
```

## Listen for Network Changes

React to user switching networks:

```typescript
import { onNetworkChanged } from 'kundera/wallet';

const unsubscribe = onNetworkChanged((event) => {
  console.log('Network changed to:', event.chainId);
  // Reconnect provider to new network
});
```

## Disconnect

```typescript
import { disconnectWallet } from 'kundera/wallet';

await disconnectWallet();
```

<Warning>
## Browser Only

Wallet connection requires `window.starknet`. These APIs only work in browser environments.
</Warning>

## React Example

```tsx
import { useState, useEffect } from 'react';
import { connectWallet, onAccountChange } from 'kundera/wallet';

function WalletButton() {
  const [address, setAddress] = useState<string | null>(null);

  useEffect(() => {
    return onAccountChange((e) => setAddress(e.address));
  }, []);

  const connect = async () => {
    const { result } = await connectWallet();
    if (result) setAddress(result.address);
  };

  return (
    <button onClick={connect}>
      {address ? `${address.slice(0, 6)}...` : 'Connect Wallet'}
    </button>
  );
}
```
