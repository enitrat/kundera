# TypeScript Quick Start

Get started with Kundera's vanilla TypeScript API.

## Installation

<CodeGroup>
```bash npm
npm install @kundera-sn/kundera-ts
```

```bash pnpm
pnpm add @kundera-sn/kundera-ts
```

```bash bun
bun add @kundera-sn/kundera-ts
```
</CodeGroup>

## Basic Usage

### Create a Provider

```typescript
import { HttpProvider } from "@kundera-sn/kundera-ts/provider"

const provider = new HttpProvider({
  url: "https://starknet-sepolia.public.blastapi.io"
})

const blockNumber = await provider.getBlockNumber()
console.log(`Latest block: ${blockNumber}`)
```

### Work with Primitives

```typescript
import { ContractAddress, Felt252 } from "@kundera-sn/kundera-ts"

// Create a contract address (validated at construction)
const address = ContractAddress("0x049d36570d4e46f48e99674bd3fcc84644ddd6b96f7c741b1562b82f9e004dc7")

// Work with field elements
const felt = Felt252("0x123")
const doubled = felt.mul(Felt252("2"))

console.log(doubled.toString()) // "0x246"
```

### Read from a Contract

```typescript
import { HttpProvider } from "@kundera-sn/kundera-ts/provider"
import { ContractAddress, Felt252 } from "@kundera-sn/kundera-ts"

const provider = new HttpProvider({
  url: "https://starknet-sepolia.public.blastapi.io"
})

const balance = await provider.callContract({
  contractAddress: ContractAddress("0x049d36570d4e46f48e99674bd3fcc84644ddd6b96f7c741b1562b82f9e004dc7"),
  entrypoint: "balanceOf",
  calldata: [
    Felt252("0x123...") // owner address
  ]
})

console.log(`Balance: ${balance[0].toString()}`)
```
