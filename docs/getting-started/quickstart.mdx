## Installation

```bash
npm install @starknet/kundera
```

Or with other package managers:

```bash
pnpm add @starknet/kundera
yarn add @starknet/kundera
bun add @starknet/kundera
```

## Choose Your Crypto Backend

Kundera needs a crypto backend for signing and hashing. Choose based on your environment:

<Tabs>
  <Tab title="Browser (WASM)">
    Load WASM at app startup:

    ```typescript
    import { loadWasmCrypto } from 'kundera/wasm';

    // Call once at startup
    await loadWasmCrypto();
    ```
  </Tab>
  <Tab title="Bun (Native)">
    Native crypto works out of the box in Bun:

    ```typescript
    import { isNativeAvailable } from 'kundera/native';

    console.log(isNativeAvailable()); // true
    ```
  </Tab>
  <Tab title="Node.js (Native)">
    Install the FFI package:

    ```bash
    npm install ffi-napi
    ```

    Then use normally:

    ```typescript
    import { isNativeAvailable } from 'kundera/native';

    console.log(isNativeAvailable()); // true if ffi-napi installed
    ```
  </Tab>
</Tabs>

## Basic Usage

### Create a Transport

```typescript
import { createHttpTransport } from 'kundera/transport';

const transport = createHttpTransport({
  url: 'https://api.zan.top/public/starknet-sepolia',
});
```

### Make RPC Calls

```typescript
import { starknet_blockNumber, starknet_call } from 'kundera/rpc';
import { ContractAddress } from 'kundera/primitives';

// Get current block
const blockNumber = await starknet_blockNumber(transport);
console.log('Block:', blockNumber);

// Call a contract
const result = await starknet_call(transport, {
  block_id: 'latest',
  request: {
    contract_address: ContractAddress('0x049d36570d4e46f48e99674bd3fcc84644ddd6b96f7c741b1562b82f9e004dc7'),
    entry_point_selector: '0x361458367e696363fbcc70777d07ebbd2394e89fd0adcaf147faccd1d294d60', // name()
    calldata: [],
  },
});
```

### Use Branded Types

```typescript
import { ContractAddress, ClassHash, Felt252 } from 'kundera/primitives';

// Create typed values
const address = ContractAddress('0x049d36570d4e46f48e99674bd3fcc84644ddd6b96f7c741b1562b82f9e004dc7');
const classHash = ClassHash('0x...');
const felt = Felt252(123n);

// Convert between formats
const hex = felt.toHex();
const bigint = felt.toBigInt();
```

### Encode/Decode Calldata

```typescript
import { encodeCalldata, decodeResult } from 'kundera/abi';

// Encode function arguments
const calldata = encodeCalldata(erc20Abi, 'transfer', [
  recipientAddress,
  { low: amountLow, high: amountHigh },
]);

// Decode return values
const balance = decodeResult(erc20Abi, 'balanceOf', result);
```

## Complete Example

Read an ERC-20 token balance:

```typescript
import { createHttpTransport } from 'kundera/transport';
import { starknet_call } from 'kundera/rpc';
import { ContractAddress, Felt252 } from 'kundera/primitives';

// Setup
const transport = createHttpTransport({
  url: 'https://api.zan.top/public/starknet-sepolia',
});

// ETH token on Starknet
const ethToken = ContractAddress('0x049d36570d4e46f48e99674bd3fcc84644ddd6b96f7c741b1562b82f9e004dc7');
const account = ContractAddress('0x...');

// balanceOf selector
const balanceOfSelector = '0x2e4263afad30923c891518314c3c95dbe830a16874e8abc5777a9a20b54c76e';

// Call
const result = await starknet_call(transport, {
  block_id: 'latest',
  request: {
    contract_address: ethToken,
    entry_point_selector: balanceOfSelector,
    calldata: [account.toHex()],
  },
});

// Result is [low, high] for u256
const low = BigInt(result[0]);
const high = BigInt(result[1]);
const balance = low + (high << 128n);

console.log('Balance:', balance);
```
