Kundera is a modern, type-safe TypeScript library for interacting with Starknet. It provides a clean, functional API for accounts, contracts, transactions, and more—designed to work seamlessly in both browser and Node.js environments.

<Tip>
Want to get started quickly? Check out [Getting Started](/getting-started) to make your first Starknet call in under 5 minutes.
</Tip>

## Simple, Intuitive API

```typescript
import { createHttpProvider } from 'kundera/provider';
import { createAccount } from 'kundera/account';
import { Felt } from 'kundera/primitives';

// Create a provider
const provider = createHttpProvider({
  url: 'https://starknet-mainnet.public.blastapi.io',
});

// Create an account
const account = createAccount({
  address: Felt.from('0x123...'),
  privateKey: Felt.from('0xabc...'),
  provider,
});

// Read from a contract
const balance = await provider.callContract({
  contractAddress: Felt.from(
    '0x049d36570d4e46f48e99674bd3fcc84644ddd6b96f7c741b1562b82f9e004dc7',
  ),
  entrypoint: 'balanceOf',
  calldata: [account.address],
});
```

## Key Features

| Feature | Description |
|---------|-------------|
| **Type Safety** | Branded types prevent mixing felts, addresses, and class hashes |
| **Tree-Shakeable** | Import only what you need for minimal bundle sizes |
| **Data-First API** | Functions take data first for easy composition and piping |
| **Error Handling** | Explicit `{result, error}` pattern—no surprise exceptions |
| **WASM Ready** | Works in browsers, Node.js, and WASM environments |
