---
title: Wallet Modal
description: Connect to wallets using StarknetKit or get-starknet modals.
---

This skill provides a minimal modal connection flow and returns a wallet provider + transport.

## Quick Start

```bash
cp -r examples/skills/wallet-modal src/skills/
```

## Usage

```ts
import { connectWalletWithModal } from './skills/wallet-modal';

const { result, error } = await connectWalletWithModal({
  modalProvider: 'starknetkit',
  chainId: 'SN_MAIN',
});

if (result) {
  const { walletProvider, transport, address } = result;
  console.log('Connected:', address);
}
```

## Notes

- This skill does not create an account abstraction.
- Pair with `account-invoke` or your own wallet-specific account flow.
- Requires peer dependencies: `@starknet-io/starknet-kit` or `get-starknet`.
