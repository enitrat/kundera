This is a quick reference for the primitives API. For detailed documentation with examples and error handling, see the [Primitives section](/typescript/primitives).

## Imports

```typescript
import {
  // Types
  Felt252,
  ContractAddress,
  ClassHash,
  StorageKey,
  EthAddress,

  // Aliases
  Felt,      // === Felt252
  Address,   // === ContractAddress
  Class,     // === ClassHash
  Storage,   // === StorageKey

  // Short strings
  encodeShortString,
  encodeShortStringHex,
  decodeShortString,
  MAX_SHORT_STRING_LENGTH,

  // Constants
  FIELD_PRIME,
  MAX_ADDRESS,
  MAX_ETH_ADDRESS,

  // Type exports
  type Felt252Type,
  type Felt252Input,
  type ContractAddressType,
  type ClassHashType,
  type StorageKeyType,
  type EthAddressType,
} from '@kundera-sn/kundera-ts';
```

## Errors

| Error | Thrown By |
|-------|-----------|
| `Invalid hex string` | `Felt252.fromHex()` |
| `Hex string too long for Felt252` | `Felt252.fromHex()` |
| `Felt252 cannot be negative` | `Felt252.fromBigInt()` |
| `Value exceeds field prime` | All Felt252 constructors |
| `Felt252 must be exactly 32 bytes` | `Felt252.fromBytes()` |
| `Contract address must be < 2^251` | `ContractAddress()` |
| `ClassHash must be < 2^251` | `ClassHash()` |
| `StorageKey must be < 2^251` | `StorageKey()` |
| `EthAddress must be < 2^160` | `EthAddress()` |
| `{str} is not an ASCII string` | `encodeShortString()` |
| `{str} is too long for short string` | `encodeShortString()` |
