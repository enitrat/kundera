Kundera exposes Starknet JSON-RPC methods as individual, tree-shakeable functions.

## Usage

```ts
import { httpTransport } from 'kundera/transport';
import { starknet_getNonce, starknet_blockNumber } from 'kundera/rpc';

const transport = httpTransport('https://api.zan.top/public/starknet-sepolia');

const block = await starknet_blockNumber(transport);
const nonce = await starknet_getNonce(transport, '0xabc');
```

## Notes

- Each `starknet_*` function accepts a `Transport` instance as its first argument.
- Import only what you use for best tree-shaking.
- For batching, use `transport.requestBatch` with `createRequest` from `kundera/transport`.
