# Kundera

**Type-safe Starknet primitives for TypeScript**

Kundera provides strongly-typed Starknet primitives, cryptography, and RPC tooling. Use it with vanilla TypeScript or with Effect-TS for composable, error-safe workflows.

## Quick Start

  <Card title="TypeScript" icon="code" href="/getting-started/typescript-quickstart">

## Choose Your Style

<Tabs>
  <Tab title="TypeScript">
    **Direct, familiar, zero extra dependencies.**

    Kundera's core library (`@kundera-sn/kundera-ts`) gives you branded types, tree-shakeable imports, and a straightforward async/await API. If you're comfortable with standard TypeScript patterns, start here.

    ```typescript
    import { HttpProvider } from "@kundera-sn/kundera-ts/provider"
    import { httpTransport } from "@kundera-sn/kundera-ts/transport"
    import { Rpc } from "@kundera-sn/kundera-ts/jsonrpc"

    const provider = new HttpProvider({
      transport: httpTransport("https://starknet-mainnet.public.blastapi.io/rpc/v0_7")
    })

    try {
      const blockNumber = await provider.request(Rpc.BlockNumberRequest())
      console.log(`Block: ${blockNumber}`)
    } catch (error) {
      console.error("RPC failed:", error)
    }
    ```
  </Tab>

  <Tab title="Effect">
    **Typed errors, dependency injection, composable services.**

    The Effect integration (`@kundera-sn/kundera-effect`) wraps the core library with Effect-TS services. Errors are tracked at the type level, dependencies are injected via layers, and resources are managed automatically.

    ```typescript
    import { ProviderService } from "@kundera-sn/kundera-effect/services"
    import { Effect } from "effect"

    const program = Effect.gen(function* () {
      const provider = yield* ProviderService
      const blockNumber = yield* provider.getBlockNumber()
      yield* Effect.log(`Block: ${blockNumber}`)
    }).pipe(
      Effect.catchAll(error =>
        Effect.logError(`Failed: ${error._tag}`)
      )
    )

    await Effect.runPromise(
      program.pipe(Effect.provide(ProviderService.layer))
    )
    ```
  </Tab>
</Tabs>

## Comparison with starknet.js

<Tabs>
  <Tab title="Kundera (TypeScript)">
    ```typescript
    import { httpTransport } from "@kundera-sn/kundera-ts/transport"
    import { getContract } from "./skills/get-contract"

    const ERC20_ABI = [
      {
        type: "function", name: "balance_of",
        inputs: [{ name: "account", type: "core::starknet::contract_address::ContractAddress" }],
        outputs: [{ type: "core::integer::u256" }],
        state_mutability: "view",
      },
    ] as const

    const contract = getContract({
      abi: ERC20_ABI,
      address: "0x049d36570d4e46f48e99674bd3fcc84644ddd6b96f7c741b1562b82f9e004dc7",
      transport: httpTransport("https://..."),
    })

    // Fully typed â€” function name autocompletes, args & return inferred
    const { result, error } = await contract.read("balance_of", [owner])
    //     ^? Uint256Type
    ```
  </Tab>

  <Tab title="Kundera (Effect)">
    ```typescript
    import { ProviderService, ContractService } from "@kundera-sn/kundera-effect/services"
    import { Effect } from "effect"

    const program = Effect.gen(function* () {
      const provider = yield* ProviderService
      const contract = yield* ContractService

      const balance = yield* contract.read({
        address: "0x...",
        entrypoint: "balanceOf",
        calldata: [owner]
      })

      return balance
    }).pipe(
      Effect.catchAll(error =>
        Effect.logError(`Failed: ${error._tag}`).pipe(
          Effect.as(0n)
        )
      )
    )

    const result = await Effect.runPromise(
      program.pipe(Effect.provide(ProviderService.layer))
    )
    ```
  </Tab>

  <Tab title="starknet.js">
    ```typescript
    import { Provider, Contract } from "starknet"

    const provider = new Provider({ nodeUrl: "https://..." })
    const contract = new Contract(abi, address, provider)

    try {
      const balance = await contract.balanceOf(owner)
      return balance
    } catch (error) {
      console.error("Failed:", error)
      return 0n
    }
    ```
  </Tab>
</Tabs>

**Key differences from starknet.js:**
- **ABI type inference** -- function names autocomplete, args and returns are inferred from ABI
- **Branded types** -- `Felt252`, `ContractAddress`, `ClassHash` are distinct types, not raw strings
- **Tree-shakeable** -- import only what you use, no monolithic bundle
- **Rust crypto** -- Pedersen, Poseidon, ECDSA via native FFI or WASM, not JS reimplementations
