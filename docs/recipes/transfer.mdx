Transfer tokens between accounts using Kundera's contract wrapper or direct account execution.

## Prerequisites

```typescript
import { createClient } from 'kundera/rpc';
import { createAccount, createSigner } from 'kundera/account';
import { getContract } from 'kundera/contract';

const client = createClient({
  url: 'https://starknet-mainnet.public.blastapi.io',
});

const signer = createSigner(process.env.PRIVATE_KEY!);
const account = createAccount(client, accountAddress, signer);
```

## Transfer ETH

ETH on Starknet is an ERC20 token at a known address:

```typescript
const ETH_ADDRESS = '0x049d36570d4e46f48e99674bd3fcc84644ddd6b96f7c741b1562b82f9e004dc7';

const eth = getContract({
  abi: ERC20_ABI,
  address: ETH_ADDRESS,
  client,
  account,
});

const { result, error } = await eth.write('transfer', [
  recipientAddress,
  1000000000000000000n, // 1 ETH (18 decimals)
]);

if (error) {
  console.error('Transfer failed:', error.message);
} else {
  console.log('TX hash:', result.transactionHash);
}
```

## Transfer ERC20 Tokens

Same pattern for any ERC20 token:

```typescript
const STRK_ADDRESS = '0x04718f5a0fc34cc1af16a1cdee98ffb20c31f5cd61d6ab07201858f4287c938d';

const strk = getContract({
  abi: ERC20_ABI,
  address: STRK_ADDRESS,
  client,
  account,
});

// Transfer 100 STRK (18 decimals)
const { result } = await strk.write('transfer', [
  recipientAddress,
  100n * 10n ** 18n,
]);
```

## Check Balance First

Verify sufficient balance before transfer:

```typescript
const { result: balance } = await eth.read('balance_of', [account.address]);

const amount = 1000000000000000000n;
if (balance[0] < amount) {
  console.error('Insufficient balance');
  return;
}

await eth.write('transfer', [recipient, amount]);
```

## Batch Transfers

Transfer multiple tokens atomically:

```typescript
const ethCall = eth.populate('transfer', [recipient, ethAmount]);
const strkCall = strk.populate('transfer', [recipient, strkAmount]);

// Both succeed or both revert
const { transaction_hash } = await account.execute([
  ethCall.result!,
  strkCall.result!,
]);
```

## Transfer with Approval

For contracts that need approval (like DEXs):

```typescript
// 1. Approve spending
const { result: approveResult } = await token.write('approve', [
  spenderContract,
  amount,
]);

// 2. Wait for confirmation (poll via getTransactionReceipt)
let receipt;
do {
  const { result } = await client.starknet_getTransactionReceipt(approveResult.transactionHash);
  receipt = result;
} while (!receipt || receipt.finality_status === 'RECEIVED');

// 3. Spender can now transfer
```

## Using Object Arguments

Named arguments for clarity:

```typescript
const { result } = await eth.write('transfer', {
  recipient: recipientAddress,
  amount: 1000000000000000000n,
});
```

## ERC20 ABI (Minimal)

```typescript
const ERC20_ABI = [
  {
    type: 'struct',
    name: 'core::integer::u256',
    members: [
      { name: 'low', type: 'core::integer::u128' },
      { name: 'high', type: 'core::integer::u128' },
    ],
  },
  {
    type: 'function',
    name: 'balance_of',
    inputs: [{ name: 'account', type: 'core::starknet::contract_address::ContractAddress' }],
    outputs: [{ name: 'balance', type: 'core::integer::u256' }],
    state_mutability: 'view',
  },
  {
    type: 'function',
    name: 'transfer',
    inputs: [
      { name: 'recipient', type: 'core::starknet::contract_address::ContractAddress' },
      { name: 'amount', type: 'core::integer::u256' },
    ],
    outputs: [{ name: 'success', type: 'core::bool' }],
    state_mutability: 'external',
  },
  {
    type: 'function',
    name: 'approve',
    inputs: [
      { name: 'spender', type: 'core::starknet::contract_address::ContractAddress' },
      { name: 'amount', type: 'core::integer::u256' },
    ],
    outputs: [{ name: 'success', type: 'core::bool' }],
    state_mutability: 'external',
  },
];
```

<Tip>
Token amounts use 18 decimals by default. Multiply human-readable amounts by `10n ** 18n`.
</Tip>
