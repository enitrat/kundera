# Transport Module

The transport layer handles raw communication with Starknet RPC nodes. Most users interact with it indirectly via [Presets](/effect/modules/presets) or [ProviderService](/effect/services/provider).

See [Transport Service](/effect/services/transport) for the full service documentation including interceptors, FiberRef policies, and custom configuration.

## Quick Reference

```typescript
import { Services } from "@kundera-sn/kundera-effect";

// HTTP transport (most common)
const httpLayer = Services.HttpTransportLive("https://my-rpc.example.com", {
  timeout: 30_000,
  headers: { "x-api-key": "..." },
});

// WebSocket transport (persistent connection)
const wsLayer = Services.WebSocketTransportLive("wss://my-rpc.example.com");

// Fiber-local policies
const program = JsonRpc.blockNumber().pipe(
  Services.withTimeout(5_000),
  Services.withRetries(3),
  Services.withRetrySchedule(Schedule.exponential("100 millis")),
);
```
